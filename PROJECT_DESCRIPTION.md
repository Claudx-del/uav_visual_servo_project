# 项目成果描述

## 项目名称

基于视觉感知的无人机智能伺服控制系统

## 项目简介

本项目构建了一套面向无人机目标跟踪任务的视觉伺服控制原型系统。系统通过摄像头获取环境图像，利用视觉检测算法识别目标位置，并将目标在图像平面中的偏差转化为无人机速度控制指令，从而实现“感知—决策—控制”的闭环控制流程。

## 技术路线

1. 图像采集：通过摄像头或视频流获取实时图像。
2. 视觉感知：采用 HSV 颜色分割检测目标区域，并输出目标中心、面积和边界框。
3. 误差建模：将目标中心与图像中心之间的偏差归一化为视觉伺服误差。
4. 控制决策：基于图像误差设计速度控制律，输出无人机前后、左右、上下方向速度。
5. 安全约束：对速度指令进行限幅，避免控制量过大。
6. 仿真验证：通过二维图像平面仿真验证目标误差收敛过程。

## 创新点

- 将计算机视觉感知结果直接嵌入无人机闭环控制过程。
- 使用模块化架构设计，视觉检测器可替换为 YOLO 等深度学习模型。
- 预留 ROS/PX4 接口，便于从仿真原型扩展到真实无人机平台。
- 可生成控制收敛曲线与实验数据，便于评估控制性能。

## 应用场景

- 无人机目标跟踪
- 智能巡检
- 视觉引导降落
- 移动目标追踪
- 复杂环境下自主飞行控制
