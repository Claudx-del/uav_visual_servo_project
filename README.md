# 基于AI视觉感知的无人机智能伺服控制系统

## 📌 项目简介

本项目构建了一套融合人工智能与控制理论的无人机视觉伺服控制系统，实现从环境感知、目标识别到闭环控制的智能化决策流程。系统通过视觉传感器获取环境信息，结合计算机视觉与数据驱动方法，实现无人机对目标的自主跟踪与稳定控制。

相较传统依赖精确数学模型的控制方法，本项目引入AI感知与自适应优化机制，在复杂动态环境中具备更好的鲁棒性与泛化能力。

---

## 🧠 技术架构

系统整体采用“感知—决策—控制”三层架构：

### 1️⃣ 视觉感知模块（AI核心）

* 基于计算机视觉算法实现目标检测与定位
* 输出目标像素坐标与相对位置信息
* 支持传统方法与深度学习模型扩展

**可扩展方向：**

* YOLOv5 / YOLOv8 目标检测
* 关键点检测 / 位姿估计
* 多目标跟踪（SORT / DeepSORT）

---

### 2️⃣ 智能决策模块（AI驱动）

* 将视觉信息映射为控制指令
* 引入数据驱动方法优化控制策略
* 支持在线或离线策略更新

**AI增强能力：**

* 控制参数自适应调整
* 误差数据驱动补偿
* 可扩展强化学习控制策略

---

### 3️⃣ 伺服控制模块

* 基于视觉误差实现闭环控制
* 支持PID及扩展控制方法
* 实现无人机对目标的稳定跟踪

---

## ⚙️ 核心功能
